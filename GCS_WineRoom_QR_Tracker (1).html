QR-Based Wine Room Tracking System

A no-scanner, phone-first tracking system using tools you already have. Staff scan a QR code on any wine shelf → a Microsoft Form opens with the item pre-filled → they log the activity → Power Automate updates the master Microsoft List automatically.

Step 1 · Microsoft Lists — Create Your Master Inventory

1

Create a new list

Go to lists.microsoft.com → New List → Blank list. Name it GCS Wine Room Inventory.

2

Add these columns

Column NameTypeNotes
TitleSingle line of textUse for SKU (e.g. GCRU7605-15) — this is the key field
WineSingle line of textFull wine name
VintageSingle line of textYear or NV
ShelfNumberShelf number
QuantityNumberCurrent bottle count
CostCurrencyCost per bottle
LastActivityDate & TimeAuto-updated by Power Automate
LastActionChoiceCheck-Out · Check-In · Audit · Tasting · Broken/Loss
LastStaffPersonWho performed last action
NotesMultiple linesOptional notes
3

Import your inventory

Use the QR Label Generator tab → export the inventory data, or manually import from the GCS_Wine_Room_Inventory.xlsx file. In Lists, click + New → Import from Excel to bulk-load all 305 SKUs.

Step 2 · Microsoft Forms — Build the Activity Form

1

Open the GCS Activity Log form

Open your existing form via the Forms link in the QR code above. Ensure it has these fields:

Field NameTypeRequired
Item ID (SKU)Short answerYes — this is pre-filled via QR URL
ActionChoiceYes — Check-Out, Check-In, Audit, Tasting, Broken/Loss
QuantityNumberYes — bottles being moved/counted
Staff NameShort answer or Name fieldYes
NotesLong answerNo
TimestampAuto-captured by FormsAuto
2

Get the pre-fill field ID for Item ID

In the form editor, click the Item ID field → click the 3 dots (…) → Pre-fill. Copy the field parameter from the URL (it looks like r329dcc1f968543...). This is already baked into your QR codes on the Label Generator tab.

3

Test one QR code

From the Label Generator tab, select any item → Print. Scan it with your phone. The form should open with the SKU pre-filled. Complete a test submission before building the automation.

Step 3 · Power Automate — Connect Form to List

1

Create a new automated cloud flow

Go to make.powerautomate.com → New Flow → Automated cloud flow. Search for trigger: When a new response is submitted (Microsoft Forms connector).

2

Get the form response details

Add action: Get response details (Microsoft Forms). Set Form ID → select your GCS Activity Log form. Set Response ID → use the dynamic value from the trigger.

3

Look up the item in Microsoft Lists

Add action: Get items (SharePoint). Set Site Address → your SharePoint. Set List Name → GCS Wine Room Inventory. Add Filter Query: Title eq '@{outputs('Get_response_details')?['body/r_ItemID']}' — replace the field name with your actual Form field ID for SKU.

4

Update the quantity with a condition

Add a Condition action. Check if Action field equals Check-Out:

  • If yes (Check-Out): Update item → set Quantity to Current Quantity - Form Quantity
  • If no (Check-In/Audit): Add another condition or directly update as appropriate

Also update LastActivity = utcNow(), LastAction = form Action value, LastStaff = form Staff Name.

5

(Optional) Send confirmation notification

Add action: Send a mobile notification or Post message in Teams channel with a summary: "✅ [Staff] checked out 2 bottles of [Wine] from Shelf [X]".

6

(Optional) Low-stock alert

After updating quantity, add a condition: if New Quantity is less than 2 → send an email/Teams alert: "⚠️ Low Stock: [Wine] — only [N] bottles remaining."

Step 4 · Generate & Print QR Labels

1

Use the Label Generator tab

Switch to the QR Label Generator tab. Search or filter by shelf. Select individual items or click Select All Visible to batch-select a shelf's worth of wines.

2

Print labels

Click Print QR Labels. A preview appears showing each label with the wine name, SKU, vintage, and shelf. Click Print — the labels are sized for standard Avery label sheets or can be cut from regular paper.

3

Attach to shelves

Print and laminate, or use adhesive label paper. Attach to the front edge of each shelf slot next to the wine. One QR code per SKU is ideal — if the same wine appears on multiple shelves, print one label per shelf location.

Tips & Best Practices

💡

QR Code URL format

Your QR codes encode: https://forms.cloud.microsoft/…?id=…&r329dcc…={SKU}. The actual SKU is substituted for each label — no {{ITEMID}} placeholder remains in the printed QRs.

💡

No login required for forms

Make sure your form is set to accept responses from anyone (not just organization members) if staff use personal phones. In Forms settings → toggle off "Only people in my organization can respond."

💡

Periodic full audits

Use the Audit action in the form to do shelf-by-shelf bottle counts. Build a separate Power Automate flow that replaces the Quantity value (instead of incrementing/decrementing) when Action = Audit.
